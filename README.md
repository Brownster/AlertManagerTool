# Prometheus Silence Manager

This is a Python application that provides a graphical interface for creating silences in one or more Prometheus instances.

## Features

- Supports creating silences for multiple Prometheus instances at once
- Allows selecting pre-created silences from a drop-down list
- Provides a text field for entering custom silence data in standard Prometheus format

## Requirements

- Python 3.6 or higher
- The `requests` library (can be installed using `pip`)

## Usage

1. Clone the repository to your local machine:

git clone https://github.com/your-username/prometheus-silence-manager.git

markdown


2. Install the `requests` library:

pip install requests

markdown


3. Start the application:

python main.py

css


4. Enter the silence data in the text field or select a pre-created silence from the drop-down list.

5. Choose the Prometheus instances to create the silence for from the drop-down list.

6. Click the "Send Silence" button to create the silence in the selected instances.
